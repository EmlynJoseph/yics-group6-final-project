Challenge 1 - Cryptography

Explanation

picoCTF challenge
Description: The one time pad can be cryptographically secure, but not when you know the key. Can you solve this? We've given you the encrypted flag, key, and a table to help UFJKXQZQUNB with the key of SOLVECRYPTO. Can you use this table to solve it?.
Solve: Step 1. Using analytical abilities we notice a sentence - “the one time pad" (based on a text).
Step 2. When we search the one time pad in the system, the system gives us an explanation. The cypher is based on a simple rule: one participant, which is a cypher, UFJKXQZQUNB (width), and a key, SOLVECRYPTO (length). Based on the concept shown above, we can tell that this encryption method is a Vigenère Cipher.
Step 3. Using the Vigenère method which equals width to length, we can tell that the decrypted code is CRYPTOISFUN. In order to get this code, we can decrypt the cypher by ourselves using the method stated previously or use this special website. Put the cipher, UFJKXQZQUNB (width), in the first textbox and the key, SOLVECRYPTO (length), in the key textbox.
The final picoCTF flag is: picoCTF{CRYPTOISFUN}
